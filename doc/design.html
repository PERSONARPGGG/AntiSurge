GAME CONCEPT

🕹
장르

뱀파이어 서바이벌 스타일 로그라이크 슈터. 끊임없이 밀려오는 바이러스 웨이브를 막으며 생존하고 성장한다.

🌆
세계관

사이버펑크 도시에 바이러스 대격변 발생. 플레이어는 해커 출신 특수요원으로서 바이러스 코어를 격리·파괴한다.

🎯
핵심 루프

킬 → 경험치 → 레벨업 카드 선택 → 무기/패시브 강화 → 보스 처치 → 저주 선택 → 반복. 스테이지 100이 최종 목표.

👥
플레이어 타깃

캐주얼~미드코어 게이머. 짧은 런(10~30분)으로 반복 플레이 가능. PC/모바일 양쪽 지원.

핵심 차별점
  • 6종 클래스 × Q 액티브 스킬 — 클래스마다 플레이스타일 완전히 다름
  • 클래스 전용 패시브 트리 — 클래스 정체성을 강화하는 전용 업그레이드
  • 무기 융합 시스템 — Lv5 무기 2개 조합 → 전설급 융합 무기
  • 저주 계약 — 리스크&리워드 의사결정으로 긴장감 유지
  • 실시간 협동 멀티플레이 (같은 기기 / Firebase 크로스 디바이스)

WEAPON SYSTEM

모든 무기는 Lv 1→5로 성장. Lv5 달성 시 레벨업 풀에 진화(Evolution) 카드 등장. 두 무기 모두 Lv5이면 융합(Fusion) 카드 등장.
무기 설명 진화형 진화 효과
⚡ 네온 플레어방사형 빛 투사체사이버 노바회전 + 폭발 확장
🔵 사이버 오비터궤도 회전 탄환양자 오비터이중 궤도 + 관통
⚡ 일렉트로 존주변 전기장 지속이온 폭풍범위 확장 + 감속
🔴 레이저 스트라이크직선 관통 레이저레이저 캐논다중 빔 + 범위↑
🪃 사이버 부메랑왕복 투사체나노 블레이드분열 + 고속
🛸 데이터 드론자동 추적 드론전투 드론드론 수↑ + 미사일
🚀 사이버 미사일유도 미사일호밍 미사일추적 강화 + 폭발↑
💍 링 블레이드확장 링 발사퀀텀 링다중 링 + 속도↑
⚡ 체인 번개적 간 연쇄 번개아크 체인연쇄 수↑ + 스턴
💣 사이버 지뢰지면 설치 폭발블랙홀 지뢰폭발 시 블랙홀 생성
🌀 블랙홀중력 끌어당기기싱귤래리티범위↑ + 흡수 피해
🔮

WEAPON FUSION — 5종

융합 무기재료 공식효과
🔮 퀀텀 스톰 네온 플레어 Lv5 + 사이버 오비터 Lv5 궤도 폭발 탄환 — 범위+피해 대폭 증가
🔮 바이러스 코어 일렉트로 존 Lv5 + 데이터 드론 Lv5 드론이 주변에 전기장 드롭
🔮 지옥불 드론 사이버 미사일 Lv5 + 데이터 드론 Lv5 드론이 유도 미사일 발사
🔮 체인 노바 링 블레이드 Lv5 + 체인 번개 Lv5 링 충돌 시 연쇄 번개 방출
🔮 보이드 마인 사이버 지뢰 Lv5 + 블랙홀 Lv5 폭발 후 블랙홀 지속 생성

BOSS PHASE SYSTEM

PHASE 1 — 100%~55%
PHASE 2 — 55%~25%
PHASE 3 — 25%~0%
  • 페이즈 1→2 (55%): 이동속도 ×1.6, 패턴 쿨다운 가속
  • 페이즈 2→3 (25%, bossIdx≥1): 이동속도 ×1.35, 모든 공격 최고 밀도
  • 보스 패턴: 돌격(charge) / 미니언 소환 / 구체 사출(orb) / 유도탄(homing) / 방어막(shield)
  • 보스 이름 10종: VIRUS ALPHA → MALWARE CORE → … → FINAL PROTOCOL

SPECIAL SYSTEMS

저주 시스템 CURSE CONTRACT

  • 보스 처치 후 저주 계약 제시 (수락 / 거절 선택)
  • 수락: 강력한 디버프 대신 즉시 강력한 보상 지급
  • 거절: 보상 없이 통과 (안전하지만 손해)
  • HP 30% 이하 근사망 시 추가 저주 자동 트리거
  • 보스 사망 시 isFinalBoss 체크 → 최종 보스는 저주 생략
