# AntiSurge ⚡

사이버펑크 네온 스타일의 생존 액션 웹게임.  
적을 처치하고, 무기를 진화시키고, 보스를 쓰러뜨려라.

**🎮 플레이:** https://personarpggg.github.io/AntiSurge/

---

## 게임 소개

끝없이 밀려오는 적들 속에서 살아남는 **자동 공격 생존 게임**.  
레벨업마다 무기와 패시브를 선택하고, 조합을 완성해 스테이지를 클리어하세요.

- 6가지 플레이어 클래스 — 각자 고유한 액티브 스킬
- 11종 무기 × 진화 시스템 — 레벨 5 달성 시 강화 형태로 진화
- 보스전 — 10스테이지마다 패턴이 다른 보스 등장
- 저주 계약, 무기 시너지, 콤보 시스템

---

## 조작법

| 입력 | 동작 |
|---|---|
| WASD / 방향키 | 이동 |
| Q | 클래스 액티브 스킬 |
| ESC / P | 일시정지 |
| 모바일 | 가상 조이스틱 (화면 좌측) |

---

## 주요 기능

- **6종 클래스**: 해커, 사이보그, 고스트, 엔지니어, 스나이퍼, 서포트
- **11종 무기 + 진화**: 플레어, 오비터, EMP 존, 레이저, 부메랑, 드론, 미사일, 링 웨이브, 바이러스 체인, 랜드마인, 블랙홀 생성기
- **보스 패턴 4종**: BERSERKER / SNIPER / SUMMONER / TITAN
- **패시브 아이템 9종**, **필드 이벤트 9종**
- **저주/축복 계약**: 보스 처치 후 위험한 거래 제안
- **무기 시너지 7종**: 조합 달성 시 자동 강화
- **메타 영구 업그레이드**, **업적 시스템**
- **세이브 슬롯 3개**, **난이도 선택** (Easy / Normal / Hard)
- **일일 도전 모드**
- **멀티플레이어** (Firebase 설정 시 크로스 기기 지원)
- PWA 지원 — 홈 화면에 추가 가능

## 기술 스택

- 순수 HTML + CSS + JavaScript (프레임워크 없음)
- HTML5 Canvas 2D 렌더링
- Web Audio API (BGM 실시간 신스 생성)
- Firebase Realtime DB (멀티플레이어, 선택사항)
- GitHub Pages 배포
