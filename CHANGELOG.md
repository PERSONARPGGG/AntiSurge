# AntiSurge — 변경 이력 (Changelog)

뱀파이어 서바이벌 스타일 사이버펑크 네온 웹게임.
각 버전은 날짜, 주요 추가 기능, 수정된 버그를 기록합니다.

---

## v0.11 — EN/KO 이중 언어 i18n + 모바일 UI 개선 (2026-06-26)

### 추가
- **EN/KO 이중 언어 지원** (`src/i18n.js`): 기본 언어 영어, 한국어 선택 가능
  - `t(key)` — UI 문자열 번역
  - `tGame(category, key, field, idx)` — 무기·패시브·미션 등 게임 데이터 번역
  - `setLang(lang)` — 언어 변경 및 localStorage 저장 (`ns_lang`)
  - `applyI18n()` — `data-i18n` 속성 일괄 DOM 적용
- **모든 무기(17종), 패시브(9종), 클래스 패시브(20종), 미션(9종), 상점(6종), 저주(4종), 영구 업그레이드(12종), 무기 융합(10종), 레전더리/부활 퍽** 영문 번역 완비
- **설정 모달에 언어 선택 토글** (EN / 한국어 버튼)
- **모바일 HUD 개선**: 스킬 버튼 좌하단 배치, 미니맵 우상단, 미션 패널 좌상단
- **미션 패널 접기/펼치기**: 기본 열림, 클릭으로 토글
- **모바일 시야 확장**: `gameZoom = 0.80` (캔버스 25% 더 넓은 영역 표시)
- **플레이어 HP 바** 캔버스 위 플레이어 위치에 항상 표시 (PC·모바일 공통)
- **중간 저장 & 이어하기**: `ns_midrun` 키로 런 중 자동 저장, 메뉴에서 복원 가능

### 수정 (버그)
- 모바일 미션 패널 HP 바 겹침 해결 (`top: 115px`으로 이동)
- 루프 버그 3건 수정 (loop.js)

## v0.09 — 검증 시스템 + Firebase 멀티플레이어 (2026-06-24)

### 추가
- **Firebase RTDB 멀티플레이어**: `FIREBASE_CONFIG` 설정 시 다른 기기와 실시간 협동 플레이
  - 플레이어 위치/HP/레벨 120ms 주기로 동기화
  - 방 코드(6자리)로 참가, `onDisconnect()` 자동 방 정리
  - `FIREBASE_CONFIG = null` 시 BroadcastChannel 폴백(같은 기기 탭)
- **자동 배포 검증 시스템** (`validate.js`): 13개 항목 정적 분석
  - 인덱스 루프 내 stage-clear 유발 패턴, 전역 배열 재할당, 모달 상태전환 누락 등
- **Git pre-commit hook** (`.githooks/pre-commit`): 커밋마다 자동 검증 실행
- **배포 스크립트** (`deploy.sh <TOKEN>`): 검증 통과 시에만 push

### 수정 (버그)
- `enemies = []` 재할당 3곳 → `.length = 0` (nuke 효과, devKillAll, devJumpStage)
- CORS manifest 경고: `file://` 프로토콜에서 `manifest.json` 로드 실패 → JS 동적 삽입으로 HTTP 서버에서만 활성화
